# 🧮 Math Speed Challenge

A simple, interactive Python command-line game that generates random arithmetic problems to test your mental math speed and accuracy.

## ✨ Features
* **Randomized Problems:** Generates unique addition, subtraction, and multiplication expressions every round.
* **Instant Validation:** Checks your answer in real-time and requires the correct result before moving to the next question.
* **Performance Timer:** Tracks exactly how long it takes you to complete the challenge.
* **Customizable Difficulty:** Easily adjust the number range or total problems within the source code.

## 🚀 How to Run
1. Ensure you have **Python 3** installed.
2. Save the script as `math_challenge.py`.
3. Open your terminal or command prompt.
4. Run the program:
   ```bash
   python math_challenge.py
